# Crypto Alert Project

This is a personal project to monitor cryptocurrency price changes and send SMS alerts using Twilio services.

## Features
- Real-time monitoring of crypto price movements
- SMS notification when price thresholds are met
- Personal educational project to learn API and notification integrations

## Technologies Used
- Python
- Flask
- Twilio API
- Crypto exchange APIs

---

This project is currently under private development and for personal use only.
